FRONT END DESIGN TRACKER — ASIC

This tracker is used to keep track of front-end progress on each FP/Block in the ASIC.
Users can see the curent state for any FP or block as well as any issues that need attention. Status for some items can be updated directly by the FP/block owners.

    • Columns shown with BLACK headings are ASIC specific data that is prefilled in by the ASIC project manager.
    • Columns shown with ORANGE headings are "auto" filled in by tools periodically analyzing the design.
    • Columns shown with BLUE headings are maintained by the FP and block owners via the dropdowns and fill-ins in the table.
    • RED indicates a problem needing attention. GREEN indicates no problems reported.
    • Dates shown below user entries indicate the last date the entry was updated.

Dropdown Instructions

RTL & DFT-RTL Status
   Integration Ready
   Check the box when your FP is ready to be integrated into the top-level netlist (including is released to the container)

   DFT RTL Status
   NA  —   Some blocks may have nothing to do (rare).
   CP  —   Coding pending. RTL coding is to be done but is not complete.
   DVP  —   DV pending. RTL coding complete, but DV of it is not.
   DVC  —   DV complete. Essentially complete and done and final resting place.		 RTL STAGE
Indicate current RTL Stage. Intent is to show the level of design content and correctness, not including review status.
NS  —   Not started.
S  —   Started, but not yet at RTL-A level.
{A, B, C, D, E, F} indicate current stage.
Timing Constraints Reviewed
NO  —   Review not completed.
IP  —   In progress. Review started but not completed.
YES  —   Review completed.		 RDL Review Status
Fill in PRRQ Number once you have one. If you have multiple PRRQ# put one in this column and others in Comments column. Double click PRRQ Number to jump to PRRQ Review.
REUSE  —   Reuse RDL, review waived.
NO  —   Not started.
IP  —   In progress in PRRQ.
APP  —   All required reviewers have approved (voted) in PRRQ.
  RTL Design Review Checklist Status
Fill in EDCS Number once you have one. Double click EDCS Number to jump to EDCS document.
NO  —   Not started.

RTL-FA  —   Review doc approved in EDCS for column F.
